()-event binding syntax is used to pass the data from 
   HTML to ts.

[]-property binding syntax is used to pass the data  
   from ts to HTML

[()]-this is called banana box syntax  which is used   
    -for two-way data binding
    -it order to use two-way data binding
	-we have to use a built-in module known as 	
	FormModule
	-Two-data binding is a very important feature of  
 	angular and very popular 
	-all form submissions is used typically done in two 
	way  binding syntax
	[(ngModel)="username"]
	where ngModel is built-in attribute given by angular. 
	username is field define in the ts field
