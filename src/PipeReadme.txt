<--------------Pipe--------------->
  
 Pipe is used to transform the data 
It is used with in template expression symbol
as name specfies pipe symbol is used 
there are following different types
1.Date
2.Currency
3.Decimal
4.Percentage
5.LowerCase
6.UpperCase
 example:
  {{percentage |percent}}
  {{name |uppercase}}
  
We can also create custom pipes in angular.
